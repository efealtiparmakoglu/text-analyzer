# Text Analyzer

Basit metin analiz araci - Python ile yazilmistir.

## Ozellikler

- ✅ Kelime sayisi hesaplama
- ✅ Karakter sayisi (bosluklu/bosluksuz)
- ✅ Cumle sayisi
- ✅ Ortalama kelime uzunlugu
- ✅ En sik kullanilan kelimeler

## Kurulum

```bash
git clone https://github.com/efealtiparmakoglu/text-analyzer.git
cd text-analyzer
pip install -r requirements.txt
```

## Kullanim

```bash
python text_analyzer.py
```
